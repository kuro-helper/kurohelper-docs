## 功能

根據關鍵字查詢公司品牌資料

## 來源

<Cards>
  <Card title="⭐批評空間" href="https://erogamescape.dyndns.org/" />
  <Card title="VNDB" href="https://vndb.org/" />
</Cards>

## 參數

| 名稱 | 說明 | 可選選項 |
|--------| ---------- | ---- |
| keyword | 要查詢的關鍵字 | // |
| 查詢資料庫選項 | 要查詢的來源 | **VNDB** |

<Callout title="warn" type="warn">查詢資料庫選項若沒選擇，會使用**批評空間**作為來源去搜尋</Callout>

![search-brand](https://image.kurohelper.com/docs/search-brand.png)

## 說明
因為批評空間是日本網站，而VNDB是歐美網站，所以使用英文查詢可以使用**VNDB**作為來源，成功率會高上不少。

![demo3](https://image.kurohelper.com/docs/search-brand-demo3.png)

⬆️使用預設來源批評空間查詢`YUZUSOFT`關鍵字就會查不到

![demo1](https://image.kurohelper.com/docs/search-brand-demo1.png)

![demo2](https://image.kurohelper.com/docs/search-brand-demo2.png)

⬆️這時候調整成VNDB就可以正常搜尋到
